# 立即 函数

[TOC]

## 定义

**函数定义 + 函数调用**

定义函数后立即执行

函数作用域 -- 无污染

```
(functino() {
    
}
)();
```

规定：立即函数后面必须加`;`

## 传参

```
(functino(a, b) {
    console.log(a+b);
}
)(2, 3);
```

## return -- 立即函数返回值，赋值给另一个变量 

```
var aa = (functino(a, b) {
    return a+b;
}
)(2, 3);

// aa == 5
```

## 特殊写法 -- 把立即函数当成表达式

把普通函数转化成立即函数
